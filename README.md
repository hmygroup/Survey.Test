## üöÄ Quick Start

1. **Prerequisites**
   - .NET 8.0 SDK
   - Visual Studio 2022 or JetBrains Rider
   - Windows 10/11

2. **Build**
   ```bash
   dotnet build Survey.Test.sln
   ```

3. **Run**
   ```bash
   dotnet run --project SurveyApp
   ```
